# 三目运算符


>样 式：3个空位，2个符号
语  法：空位 ？ 空位 ：空位;
关键信息：bool类型 ？ bool类型为真返回内容 : bool类型为假返回的内容;
三目运算符会有返回值，这个返回值类型必须一致，并且必须使用

<!-- more -->   

使用：
```csharp
string a = false ? "条件为真" : "条件为假";
Console.WriteLine(a);
```

---

```csharp
int a = 5;
string str;
str = a < 1 ? "真" : "假";
Console.WriteLine("5<1是"+str+"的");
```
---

>第一个空位始终是结果为bool类型的表达式，bool变量 条件表达式 逻辑运算符表达式
第二三个空位 可以写其他表达式 保证结果类型是一致的

如以下例子：

```csharp
int a = 5;
int b = a > 10 ? 123 : 456;//注意这里是int 
Console.WriteLine(b);

结果：456
```

---


```csharp
int a = 5;
bool b = a > 1 ? !false : 1 + 1 > 10;//注意这里是bool
Console.WriteLine(b);

结果：True
```
