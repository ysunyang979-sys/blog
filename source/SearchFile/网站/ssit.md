<a href="https://roadmap.sh/dashboard" class="btn">Roadmap</a>学习路线
<a href="https://fmhy.net" class="btn">Free Media Heck Yeah</a>互联网上最大的免费资源导航百科，包含影视、软件、书籍等各类资源。
<a href="https://theresanaiforthat.com/ai/thea/#pricing-options" class="btn">There's An AI For That</a>极其全面的 AI 工具数据库，可以根据需求搜索几乎所有现有的 AI 应用。
<a href="https://wotaku.wiki/art" class="btn">Wotaku Art</a>专为艺术家和创作者设计的资源百科，涵盖绘画教程、参考资料和工具。
<a href="https://free-for.dev/#/?id=apis-data-and-ml" class="btn">Free-for.dev</a>专门为开发者整理的免费服务列表（如 API、托管、数据科学工具等）。
<a href="https://alternativeto.net" class="btn">AlternativeTo</a>软件替代方案搜索平台，帮你寻找某款收费或不好用软件的免费/开源替代品。
<a href="https://www.openculture.com" class="btn">Open Culture</a>汇集全球免费的文化与教育资源，包括在线课程、有声书、电影和电子书。
<a href="https://tinywow.com/tools/file_conversion" class="btn">TinyWow</a>强大的在线工具合集，免费处理 PDF、视频、图像转换及各类文件编辑。
<a href="https://www.manualslib.com" class="btn">ManualsLib</a>全球最大的产品说明书数据库，可以找到各种电子产品和家电的使用手册。
<a href="https://tosdr.org/zh" class="btn">Terms of Service; Didn't Read</a>将各大网站冗长的“服务条款”总结成简单易懂的分级评价，保护用户隐私。
<a href="https://www.myfonts.com" class="btn">MyFonts</a>全球知名的字体市场，支持通过图片识别未知字体。
<a href="https://ourworldindata.org" class="btn">Our World in Data</a>用数据研究全球性问题（如贫困、健康、气候）的权威网站，图表化呈现。
<a href="https://devdocs.io" class="btn">DevDocs</a>开发者必备的文档浏览器，在一个界面下查阅上百种编程语言和框架的文档。
<a href="https://learnxinyminutes.com" class="btn">Learn X in Y minutes</a>快速入门编程语言的代码笔记，让你在几分钟内掌握某种语言的核心语法。
<a href="https://www.virustotal.com/gui/home/upload" class="btn">VirusTotal</a>著名的在线杀毒扫描工具，同时调用几十种杀毒引擎检测文件或链接是否安全。
<a href="https://thenounproject.com" class="btn">The Noun Project</a>高质量的矢量图标和摄影照片库，设计风格简约统一。
<a href="https://haveibeenpwned.com" class="btn">Have I Been Pwned</a>安全查询网站，输入邮箱即可查看你的账号是否在已知的数据泄露事件中被曝光。
<a href="https://www.omnicalculator.com" class="btn">Omni Calculator</a>万能计算器集合，包含数学、财务、健康、生活等数千个细分领域的计算器。
<a href="https://wormhole.app" class="btn">Wormhole</a>极速、私密且端到端加密的超大文件传输工具，链接限时即焚。
<a href="https://asoftmurmur.com" class="btn">A Soft Murmur</a>环境白噪音生成器，可以混合雨声、雷声、风声等帮助专注或入眠。
<a href="https://skillsmp.com/" class="btn">SkillSMP</a>技能学习与实践平台，旨在通过结构化的方式提升个人专业能力。
