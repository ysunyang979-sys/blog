# 💻 Windows 极致装机指南

> **“极简、提效、自主掌控”**
> 一个优秀的 Windows 系统不应充斥着广告与全家桶。我整理了一份深度清单，用开源、轻量的工具重塑你的数字工作流。

### 🧰 核心工具体系
*   **📦 包管理器 (UniGetUI/Scoop)** ：像手机应用商店一样管理 PC 软件，一键安装、静默升级，彻底告别流氓捆绑。
*   **📂 文件闪搜 (Everything/Listary)** ：毫秒级全盘检索，让“查找”不再中断你的思路。
*   **⚡ 指尖效率 (Snipaste/Quicker)** ：截图、贴图、快捷指令，将繁琐的连招简化为一次点击。
*   **🛡️ 纯净维护 (火绒/Geek Uninstaller)** ：只拦截不打扰，强力卸载残留，保持系统长久如新。
