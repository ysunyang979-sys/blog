异常捕获

作用：可以避免当代码报错时造成程序卡死的情况

try
{
    Concle.WriteLine("异常捕获");
    string str = Console.ReadLine();
    int i = int.Parse(str);
}
catch
{
    Console.WriteLine("输入的有误"); //如果try中的代码有误则会执行这里的代码
}
finally
{
    Consle.WriteLine("执行完毕"); //无论对错都会执行这里的代码
}

练习：用户输入一个数字，如果有误则提示用户输入错误

  try
  {
      string str1 = Console.ReadLine();
      int i1 = int.Parse(str1);
      Console.WriteLine("输入正确");
  }
  catch
  {
      Console.WriteLine("请输入正确的数字");
  }
